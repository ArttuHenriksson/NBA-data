NBA Stats api

Pelaajien tilastojen hakemiseen käytin NBA Stats API. Tilastot ovat vuosilta 2006-2023.

Angular Material

Käytin Angular Material komponenttikirjastoa tässä lopputyössä. Käytin table komponettia pelaajien tilastojen esittämistä varten.

Myös käytin matTooltippiä topRebounds komponentissa selkeyttämään eri levypallo tilastojen nimiä.

PrimeNG

Käytin myös toista komponenttikirjastoa nimeltään PrimeNG. Sieltä käytin kahta valmista komponettia Accordionia sekä divideria.

Divideriä käyttän tässä sivulla ja accordionia käytän pelaajien hakemisessa.

NGX Spinner

Latausspinnereihin käytin NgxSpinner kirjastoa.

Tailwind

Käytin tyylittelyyn tässä työssä Tailwindia.
